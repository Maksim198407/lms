# LMS — Система управления обучением

## Быстрый старт

### Требования
- Python 3.10+

### Запуск

```bash
# Windows — двойной клик на run.bat или:
run.bat

# Или вручную:
cd backend
pip install -r requirements.txt
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

### Доступ
- **Веб-интерфейс**: http://localhost:8000
- **API документация**: http://localhost:8000/docs

### Учётные данные по умолчанию
- **Email**: `admin@lms.local`
- **Пароль**: `admin123`

## Архитектура

```
lms/
├── backend/
│   ├── app/
│   │   ├── main.py          # FastAPI приложение
│   │   ├── config.py        # Настройки
│   │   ├── database.py      # SQLAlchemy подключение
│   │   ├── auth.py          # JWT авторизация
│   │   ├── models/
│   │   │   └── models.py    # ORM модели
│   │   ├── schemas/
│   │   │   └── schemas.py   # Pydantic схемы
│   │   └── routers/
│   │       ├── auth.py       # Авторизация
│   │       ├── users.py      # Пользователи
│   │       ├── courses.py    # Курсы
│   │       ├── groups.py     # Группы
│   │       ├── themes.py     # Темы
│   │       ├── categories.py # Категории
│   │       ├── assignments.py# Задания и тесты
│   │       └── dashboard.py  # Статистика и экспорт
│   ├── uploads/              # Загруженные файлы
│   └── requirements.txt
├── frontend/
│   ├── index.html
│   ├── css/style.css
│   └── js/
│       ├── api.js            # HTTP клиент
│       ├── router.js         # SPA маршрутизация
│       ├── auth.js           # Страницы авторизации
│       ├── admin.js          # Панель администратора
│       ├── student.js        # Интерфейс студента
│       └── app.js            # Инициализация
└── run.bat
```

## Возможности

### Администратор
- Управление курсами, темами, группами
- Создание тестов с вопросами и вариантами ответов
- Создание заданий с документами (PDF, DOC, XLS, PPT)
- Просмотр и изменение оценок
- Назначение повторных попыток
- Панель статистики по группам и студентам
- Экспорт оценок в Excel
- Уведомления студентов

### Студент
- Просмотр курсов и тем
- Прохождение тестов с автоматической оценкой
- Загрузка файлов с ответами
- Просмотр оценок
- Уведомления о заданиях

## API

Полная документация API доступна по адресу `/docs` (Swagger UI).
